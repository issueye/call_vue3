# 图标说明

## Windows
- `icon.ico` - 256x256 PNG 转 ICO 格式
- `iconSmall.ico` - 16x16 图标
- `iconBig.ico` - 48x48 图标

## macOS
- `icon.icns` - 1024x1024 PNG 转 ICNS 格式

## Linux
- `icon.png` - 256x256 PNG 格式

## 生成工具

### Windows 图标
可以使用以下工具生成：
- https://icoconvert.com/
- GIMP

### macOS 图标
```bash
# 使用 png2icns (macOS)
png2icns icon.icns icon.png

# 或使用在线工具
# https://icoconvert.com/
```

### Linux 图标
直接使用 256x256 PNG 格式即可
